## ⚡ Core Architecture & Features

This platform is not a standard template. It is a custom-engineered web application demonstrating clean architecture, component isolation, and advanced frontend physics.

- **App Router (Next.js 15):** Leveraging React Server Components (RSC) to ship zero JavaScript where possible.
- **Incremental Static Regeneration (ISR):** Integrates live GitHub API data (contribution graphs, repo stats) without hitting client-side rate limits.
- **MDX Blog Engine:** Fully integrated markdown parser with `next-mdx-remote` and VS Code-grade syntax highlighting (`rehype-pretty-code`).
- **Advanced Physics & Motion:** Buttery smooth scroll transitions (Lenis), bento-box grid layouts, layout re-ordering, and an interactive custom cursor using `framer-motion`.
- **Command Palette Integration:** Hit `⌘K` (or `Ctrl+K`) to instantly navigate the application (powered by `cmdk`).
- **SEO & Open Graph:** Dynamically generated `sitemap.xml`, `robots.txt`, and edge-rendered `opengraph-image.png`.

## 🚀 Local Development

Follow these steps to run the application locally on your machine.

**1. Clone the repository**
```bash
git clone https://github.com/shiv123-coder/Shivshankar_Mali_Portfolio.git
cd Shivshankar_Mali_Portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Configure Environment Variables**
Copy the `.env.example` to `.env.local`:
```bash
cp .env.example .env.local
```

**4. Start the development server**
```bash
npm run dev
```

Visit `http://localhost:3000` to view the application.
